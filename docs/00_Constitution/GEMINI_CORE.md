# GEMINI CORE: Global Marathon Hub

이 문서는 Global Marathon Hub 프로젝트의 모든 에이전트가 준수해야 할 프로젝트 전용 헌법입니다.

1. **기록 불변:** 모든 대화 세션은 `docs/99_Logs/`에 증분 파일로 기록한다.
2. **비용 효율:** 초기 모든 인프라는 무료 티어(Zero-Cost)를 우선하며, 확장성을 고려한 추상화 레이어를 둔다.
3. **시각 기반:** 데이터 소스 분석 시 반드시 스크린샷 촬영 및 시각 데이터 분석을 선행한다.
4. **글로벌 대응:** 모든 데이터 구조는 다국어(i18n) 및 국제 표준 시(UTC) 대응이 가능하도록 설계한다.
