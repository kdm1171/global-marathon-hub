# PRD: Global Marathon Hub (Korea First)

## 1. 제품 정의
대한민국의 모든 마라톤 대회(메이저~지방 소규모)를 가장 트렌디하고 편리하게 탐색할 수 있는 모바일 최적화 웹 서비스.

## 2. 핵심 페르소나 & 가치
- **러닝 입문자:** "접수 마감 전에 알려줘!" (일정 시각화)
- **전국 투어 러너:** "이번 주말 강원도에 대회 없나?" (지역 필터)
- **데이터 분석가:** "이 코스 오르막이 얼마나 심해?" (고도/코스 지도)

## 3. 주요 기능 (MVP)
### 3.1. 스마트 마라톤 캘린더
- 월별/주별 대회 일정 뷰.
- **접수 중 / 접수 예정 / 접수 마감** 상태를 직관적인 배지로 표시.

### 3.2. 정밀 검색 및 필터
- 지역(도/시/군) 기반 검색.
- 거리(5km, 10km, 하프, 풀) 기반 필터.

### 3.3. 상세 정보 및 연동
- 대회장 출발 지점 지도 표시.
- **Google Calendar / iCal 일정 자동 추가**.

## 4. 기술적 합의 (The 0-Won Engine)
- **Infrastructure:** Vercel (Hobby).
- **Data Pipeline:** GitHub Actions Cron + Playwright Scraper.
- **SEO:** 대회명/지역명을 활용한 다이나믹 메타 태그 생성.
